# Akkadian DSL Support

This extension provides language support for **Akkadian DSL v3.0**, the core language of the **beeMDM** ecosystem.

![beeMDM](beeMDM3_5.webp)
## Features

- **Syntax Highlighting**: Colorization for Contexts, Identities, Data Vault structures, and Vectorization blocks.
- **Snippets**: Quickly generate Hubs, Satellites, and Commands using `hub`, `sat`, `cmd` triggers.
- **Structure**: Visual distinction for the Akkadian Grammar.

## Usage

Create a file with the `.akk` extension to see the highlighting in action.
